# Setup

## Install ***rustup***

Run the following command:
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Enable ***nightly*** mode
```
rustup default nightly
```

## Spin it up!

```
cargo run
```

Congratulations, your Rust server is running!
